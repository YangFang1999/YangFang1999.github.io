关于我

你好！我是一名正在学习 Java 的后端开发新手。之前有一些 C 语言基础，现在决定专注于 Java 生态的学习。

我创建这个博客的目的是为了记录我的学习过程，把遇到的问题和解决方案分享出来。一方面是为了巩固自己的知识，另一方面也希望能帮助到和我一样的初学者。

我的学习路线是从 Java SE 开始，然后逐步学习 Spring、Spring Boot 等主流框架，最终目标是成为一名合格的 Java 后端工程师。
