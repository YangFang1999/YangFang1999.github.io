Java 技术栈概览：从入门到实战

发布于 2024-06-01 · 预计阅读 8 分钟

快速实践建议：从一个最小的 REST API 开始（Spring Boot + JPA + MySQL），为 2-3 个实体设计 CRUD；补上单元测试与简单日志；将经验沉淀为一篇复盘笔记。

这篇文章汇总了 Java 常用的技术栈内容——覆盖语言基础、常用框架、数据库与持久化、构建与部署、测试与监控等模块。帮助初学者快速建立整体视图，明确下一步学习方向。

1. 语言基础与核心库

  • Java 语法基础：变量、流程控制、面向对象（封装/继承/多态）。
  • 集合框架：List/Set/Map 及常见实现（ArrayList、HashSet、HashMap）。
  • 并发编程：线程、锁、并发工具（Executor、Future、CompletableFuture）。
  • I/O 与 NIO：文件处理、缓冲、通道、选择器。
  • JVM 基础：类加载、内存模型、垃圾回收。

2. Spring & Spring Boot

  • 核心概念：IoC、AOP、依赖注入与切面。
  • Spring MVC：RESTful API、参数绑定、全局异常处理。
  • Spring Boot：自动配置、Starter、配置管理（application.yml）。
  • 常用组件：Spring Data JPA、Spring Security、Validation。

3. 数据库与持久化

  • 关系型数据库：MySQL 基础、索引、事务与锁。
  • ORM 框架：JPA/Hibernate 基本映射、查询、性能优化。
  • NoSQL：Redis 常用数据结构与缓存策略。
  • 数据库迁移：Flyway/Liquibase。

4. 构建与依赖管理

  • Maven/Gradle：项目结构、依赖、构建生命周期。
  • 打包与版本：可执行 JAR、版本规范、语义化版本。
  • 简单部署：GitHub Pages（前端）、使用 Render/ Railway 部署后端 Demo。

5. 测试与质量

  • 单元测试：JUnit 5、Mockito。
  • 集成测试：Spring Boot Test、Testcontainers（了解）。
  • 代码质量：Checkstyle、SpotBugs、SonarLint。

6. 监控与可观测性（入门）

  • 日志：SLF4J + Logback；结构化日志。
  • 指标：Micrometer + Prometheus（了解）。
  • 链路：Spring Cloud Sleuth（了解）。

7. 学习路线建议

  1. Java SE 语法 + 集合 + 并发入门。
  2. Spring MVC 构建 REST API，配合数据库（MySQL + JPA）。
  3. 引入 Spring Boot，完善配置与组件，加入测试。
  4. 做一个小项目：例如 Todo/书签管理/学习笔记 API 等。
  5. 了解部署与监控，形成完整闭环。
小贴士：学习过程中，记录自己的问题与解决方案，定期复盘，这样能形成自己的知识体系。
