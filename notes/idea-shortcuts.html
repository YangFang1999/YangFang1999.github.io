IntelliJ IDEA 新手常用快捷键整理

2024-05-15 阅读时间: 约 5 分钟

大家好！在上一篇笔记中，我们成功运行了第一个 Java 程序。工欲善其事，必先利其器。今天，我想和大家分享一些我在使用 IntelliJ IDEA (简称 IDEA) 过程中收集和整理的常用快捷键。熟练使用这些快捷键，可以让你的编码速度和效率大大提升。

注意：以下快捷键以 Windows / Linux 系统为准。Mac 用户通常可以将 Ctrl 键替换为 Cmd 键。

1. 编辑操作 (最常用)

  • Ctrl + Space：代码自动补全。
  • Ctrl + D：复制当前行或选中的代码块到下一行。
  • Ctrl + Y：删除当前行。
  • Ctrl + /：单行注释 / 取消注释。
  • Ctrl + Shift + /：块注释 / 取消注释。
  • Ctrl + Alt + L：格式化代码。

2. 代码导航

  • Ctrl + B 或 Ctrl + 鼠标左键点击：跳转到定义。
  • Alt + ← / →：在不同位置之间切换。
  • Ctrl + Shift + F：全局搜索文本。

3. 重构与优化

  • Shift + F6：重命名。
  • Ctrl + Alt + O：优化导入。

4. 运行与调试

  • Shift + F10：运行。
  • Shift + F9：调试。
  • F8：Step Over。
  • F7：Step Into。

5. 其他实用功能

  • Alt + Insert：生成代码。
  • Ctrl + Tab：切换文件。
  • Ctrl + F4：关闭当前标签。
