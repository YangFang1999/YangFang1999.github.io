Java 入门：Hello World 详解与环境配置

2024-05-20 阅读时间: 约 8 分钟

大家好！这是我的第一篇 Java 学习笔记。作为一个不折不扣的新手，我想把我从零开始的每一步都记录下来，希望能帮助到和我一样刚刚起步的朋友。今天，我们就从最经典的 "Hello, World!" 程序开始。

1. 准备工作：安装 JDK

要运行 Java 程序，首先得有 Java 开发环境，也就是 JDK (Java Development Kit)。

  1. 下载 JDK：我推荐从 Oracle 官网 下载。注意，现在主流的是 Java 17 或 21。选择对应操作系统的安装包。
  2. 安装：运行安装程序，一路点击下一步即可。建议不要修改默认安装路径，或者记下你安装的路径，后面会用到。

2. 配置环境变量 (Windows)

这是最关键也最容易出错的一步。

  1. 右键点击 "此电脑" -> "属性" -> "高级系统设置" -> "环境变量"。
  2. 在 "系统变量" 区域，点击 "新建"。
    • 变量名：JAVA_HOME
    • 变量值：你的 JDK 安装路径，例如 C:\Program Files\Java\jdk-17
  3. 找到名为 Path 的系统变量，点击 "编辑"。
  4. 点击 "新建"，然后输入 %JAVA_HOME%\bin，点击确定。
  5. 再次点击 "新建"，输入 %JAVA_HOME%\jre\bin (如果是 Java 9 及以上版本，可能没有 jre 文件夹，可以省略这一步)，点击确定。
  6. 一路点击确定关闭所有窗口。

验证：打开一个新的命令提示符（CMD）或 PowerShell，输入以下命令，如果能正确显示版本号，就说明配置成功了。

java -version
javac -version

3. 使用 IntelliJ IDEA 编写第一个程序

虽然可以用记事本写代码，但强烈建议使用一个 IDE (集成开发环境)，能大大提高效率。我使用的是 IntelliJ IDEA Community Edition (社区版，免费)。

  1. 创建项目：打开 IDEA，选择 "New Project"。选择 "Java"，然后选择你刚刚配置好的 JDK，点击 "Next"，再点击 "Next"。给项目起个名字，比如 HelloWorldProject，点击 "Finish"。
  2. 创建 Java 类：在左侧项目视图中，右键点击 src -> "New" -> "Java Class"。
    • Class name：输入 HelloWorld (注意大小写，Java 类名首字母大写)。
    • Kind：选择 Class。
    • 点击 "Enter"。
  3. 编写代码：在打开的 HelloWorld.java 文件中，输入以下代码：
public class HelloWorld {
    public static void main(String[] args) {
        // 这是我的第一个 Java 程序
        System.out.println("Hello, World!");
    }
}

代码解释：

  • public class HelloWorld：定义了一个名为 HelloWorld 的公共类。文件名必须和类名一致。
  • public static void main(String[] args)：这是 Java 程序的入口点，也就是main方法。程序从这里开始执行。
  • // 这是我的第一个 Java 程序：这是单行注释，编译器会忽略它。
  • System.out.println("Hello, World!");：这行代码的作用是在控制台打印出 "Hello, World!" 这句话。; 表示语句结束。

4. 运行程序

在 IDEA 中运行程序非常简单：

  1. 在代码编辑区，右键点击任意位置。
  2. 选择 "Run 'HelloWorld.main()'"。

你应该能在下方的 "Run" 控制台看到输出结果：

Hello, World!

恭喜你！你已经成功编写并运行了你的第一个 Java 程序！

5. 常见问题排查

  • "java" 不是内部或外部命令...：这通常是环境变量 Path 配置错误。请仔细检查第 2 步。记住，修改环境变量后需要打开新的 CMD 窗口才能生效。
  • 文件名和类名不匹配：如果你的类名是 HelloWorld，你的文件名必须是 HelloWorld.java，否则编译器会报错。
  • 大小写错误：Java 是大小写敏感的。system.out.println 和 System.out.println 是不一样的，后者才是正确的。

总结

今天我们完成了 Java 开发的入门，包括安装 JDK、配置环境变量、使用 IDEA 创建项目、编写并运行 "Hello, World!" 程序。这是一个非常好的开始。接下来，我们将继续学习 Java 的基本语法。

如果你在过程中遇到了任何问题，欢迎在评论区留言，我们一起讨论解决！
