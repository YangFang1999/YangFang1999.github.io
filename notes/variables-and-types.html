Java 中的变量、常量与数据类型详解

2024-05-10 阅读时间: 约 10 分钟

Hello, 各位小伙伴！在编写了第一个 "Hello, World!" 程序后，我们来深入了解一下 Java 程序的基本构成单元：变量和数据类型。这是任何编程语言的基石，理解它们对于后续的学习至关重要。

1. 变量 (Variable)

变量就像一个容器，用来存储数据。这个容器有一个名字（变量名），一个类型（数据类型），并且可以存放一个值。在程序运行过程中，变量的值是可以改变的。

变量的定义

在 Java 中，定义一个变量的语法是：

// 数据类型 变量名;
// 数据类型 变量名 = 初始值;

int age;             // 定义了一个名为 age 的整数类型变量，此时它没有值
age = 25;            // 给变量 age 赋值为 25

String name = "张三"; // 定义并初始化一个名为 name 的字符串变量

变量命名规则

给变量起名字时需要遵循一些规则：

  1. 可以由字母（A-Z, a-z）、数字（0-9）、下划线（_）和美元符号（$）组成。
  2. 不能以数字开头。
  3. 不能是 Java 的关键字（如 int, class, public 等）。
  4. 区分大小写，age 和 Age 是两个不同的变量。
  5. 建议使用有意义的名称，遵循 "驼峰命名法"（camelCase），即第一个单词首字母小写，后面单词首字母大写，例如 userName, studentId。

2. 常量 (Constant)

常量类似于变量，但它的值在程序运行期间**不能被改变**。它是固定不变的。

常量的定义

在 Java 中，使用 final 关键字来定义常量：

// final 数据类型 常量名 = 值;

final double PI = 3.1415926;
final String SCHOOL_NAME = "北京大学";

// PI = 3.14; // 这行代码会报错，因为常量的值不能被修改

常量命名通常使用全大写字母，单词之间用下划线分隔。

3. 数据类型 (Data Type)

Java 是一种强类型语言，这意味着在定义变量时，必须明确指定它的数据类型。数据类型决定了变量可以存储什么样的值，以及可以对其进行什么样的操作。

Java 的数据类型主要分为两大类：

基本数据类型 (Primitive Types)

基本数据类型是 Java 语言内置的、不可再分的类型。它们直接存储值。共有 8 种：

  • 整数类型：
    • byte：1 字节，范围 -128 ~ 127
    • short：2 字节，范围 -32768 ~ 32767
    • int：4 字节，范围 -2147483648 ~ 2147483647 (最常用)
    • long：8 字节，范围很大。定义时需在数值后加 L 或 l，如 long num = 10000000000L;
  • 浮点类型（小数）：
    • float：4 字节，单精度浮点数。定义时需在数值后加 F 或 f，如 float price = 9.9f;
    • double：8 字节，双精度浮点数 (最常用)
  • 字符类型：
    • char：2 字节，用于存储单个字符，用单引号括起来，如 char c = 'A'; 或 char chineseChar = '中';
  • 布尔类型：
    • boolean：1 字节，只有两个值：true (真) 和 false (假)，用于表示逻辑判断

引用数据类型 (Reference Types)

引用数据类型存储的是对象的"引用"（可以理解为对象在内存中的地址），而不是对象本身。

  • 字符串：String (注意 S 是大写)。用于存储一串字符，用双引号括起来，如 String greeting = "Hello, Java!";。String 是一个类，不是基本类型。
  • 数组：int[] arr = new int[10];
  • 类 (Class)：自定义的类或 Java 提供的其他类，如 Scanner, ArrayList 等。

4. 类型转换 (Type Casting)

在某些情况下，我们需要将一种数据类型的值转换为另一种数据类型。

自动类型转换（隐式转换）

当两种类型兼容，且目标类型的取值范围大于源类型时，Java 会自动进行转换。

int i = 100;
long l = i; // int -> long，自动转换，因为 long 范围更大
double d = l; // long -> double，自动转换

强制类型转换（显式转换）

当目标类型的取值范围小于源类型时，或者类型不兼容时，需要进行强制转换。这可能会导致精度损失或数据溢出。

double pi = 3.1415;
int num = (int) pi; // 强制转换，结果 num = 3，小数部分被舍弃

long bigNum = 2147483648L; // 这个值超过了 int 的最大值
int smallNum = (int) bigNum; // 强制转换，会发生溢出，结果可能是负数或其他意想不到的值

注意：boolean 类型不能与其他任何类型进行转换。

总结

今天我们学习了 Java 中变量、常量的定义和使用，以及数据类型的分类。理解基本数据类型和引用数据类型的区别，以及掌握类型转换的规则，是写出正确、高效代码的基础。
